nDLA 0.1.0.0

N-dimensional Linear Algebra in classless C++.

This project began as The Euclidean Project (https://github.com/tarbalReboot/euclideanSpace/) by the same sole author. The goal of this project is to generalize several of the R2, R3, and R4 linear algebra routines in Euclidean to MxN-dimensionality. This goal is quite a bit more complex, so the function calls are much more difficult to master, as a result. 

Included is an example.cpp file, as a crude example of how to allocate a memory buffer for Echelon row reduction, etc. This package is in its infancy, and even the author does not know how far it will go.
